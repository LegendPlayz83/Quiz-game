# Quiz-game <br>
This is an HTML Quiz Game which involves 4 different genres <br>
Each genre has 5 individual questions <br>
The genres are: <ul>
<li>Taylor Swift</li>
<li>Bollywood</li>
<li>Science</li>
<li>Geography</li> <br>

<h3>Project contents: </h3>
The project has mainly 3 files. The files are: <ul>
<li>index.html</li>
<li>script.js</li>
<li>style.css</li> <br>

The score is determined after all questions from a genre have been answered
